# Project Plan

This plugin's near-term roadmap:

- **Auth**: PAT-only (see `SOURCE_CONTROL_PROVIDERS.md` for why OAuth was
  removed) — stable, not expected to change.
- **Source-control providers**: GitHub is the only implemented provider
  today. The seam for adding GitLab/Bitbucket exists (see
  `SOURCE_CONTROL_PROVIDERS.md`) but is intentionally deferred until there's
  real demand for a second provider.
- **Shipped history**: see `CHANGELOG.md`.

This file is intentionally short — it exists so the "Project Plan" link
from `CONTRIBUTING.md` resolves to something real. Expand it as the roadmap
firms up.
