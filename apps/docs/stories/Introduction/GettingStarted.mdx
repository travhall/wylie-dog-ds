# Getting Started

Get up and running with the Wylie Dog Design System in minutes.

## üì¶ Installation

Install the component library using your preferred package manager:

```bash
# Using pnpm (recommended)
pnpm add @wyliedog/ui

# Using npm
npm install @wyliedog/ui

# Using yarn
yarn add @wyliedog/ui
```

## ‚öôÔ∏è Setup

### 1. Import Global Styles

Import the design system's global CSS in your app's entry point:

```tsx
// app/layout.tsx or main.tsx
import '@wyliedog/ui/globals.css';

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

### 2. Configure Tailwind (Optional)

If you're using Tailwind CSS in your project, extend our configuration:

```js
// tailwind.config.js
import { tailwindConfig } from '@wyliedog/ui/tailwind.config';

export default {
  ...tailwindConfig,
  content: [
    './src/**/*.{js,ts,jsx,tsx}',
    './node_modules/@wyliedog/ui/**/*.{js,ts,jsx,tsx}',
  ],
};
```

### 3. TypeScript Configuration (Optional)

For optimal TypeScript support, ensure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "moduleResolution": "bundler",
    "strict": true
  }
}
```

## üéØ Common Patterns

### Using Design Tokens

Components automatically use design tokens, but you can also use them directly:

```tsx
// Using Tailwind classes
<div className="bg-primary-500 text-white p-4 rounded-lg">
  Primary colored box
</div>

// Using CSS variables
<div style={{ backgroundColor: 'var(--color-primary-500)' }}>
  Custom styled element
</div>
```

### Composing Components

Components are designed to work together seamlessly:

```tsx
import { Card, CardHeader, CardTitle, CardContent } from '@wyliedog/ui/card';
import { Button } from '@wyliedog/ui/button';
import { Badge } from '@wyliedog/ui/badge';

export function UserCard() {
  return (
    <Card className="w-80">
      <CardHeader>
        <div className="flex items-center justify-between">
          <CardTitle>John Doe</CardTitle>
          <Badge variant="success">Active</Badge>
        </div>
      </CardHeader>
      <CardContent className="space-y-4">
        <p className="text-sm text-neutral-600">
          Senior Frontend Developer
        </p>
        <div className="flex gap-2">
          <Button variant="primary" size="sm">
            Message
          </Button>
          <Button variant="outline" size="sm">
            View Profile
          </Button>
        </div>
      </CardContent>
    </Card>
  );
}
```

### Handling States

Components support various states for different use cases:

```tsx
import { Alert, AlertTitle, AlertDescription } from '@wyliedog/ui/alert';

export function Notifications() {
  return (
    <div className="space-y-4">
      {/* Success state */}
      <Alert variant="success">
        <AlertTitle>Success!</AlertTitle>
        <AlertDescription>
          Your changes have been saved.
        </AlertDescription>
      </Alert>

      {/* Warning state */}
      <Alert variant="warning">
        <AlertTitle>Warning</AlertTitle>
        <AlertDescription>
          This action cannot be undone.
        </AlertDescription>
      </Alert>

      {/* Error state */}
      <Alert variant="destructive">
        <AlertTitle>Error</AlertTitle>
        <AlertDescription>
          Something went wrong. Please try again.
        </AlertDescription>
      </Alert>
    </div>
  );
}
```

## ‚ôø Accessibility

All components are built with accessibility in mind:

```tsx
import { Button } from '@wyliedog/ui/button';
import { Input } from '@wyliedog/ui/input';
import { Label } from '@wyliedog/ui/label';

export function AccessibleForm() {
  return (
    <form>
      {/* Always pair inputs with labels */}
      <Label htmlFor="username">Username</Label>
      <Input
        id="username"
        aria-describedby="username-help"
        required
      />
      <p id="username-help" className="text-sm text-neutral-600">
        Choose a unique username
      </p>

      {/* Use semantic button types */}
      <Button type="submit" aria-label="Submit form">
        Submit
      </Button>
    </form>
  );
}
```

## üé® Customization

### Custom Styling

Extend component styles with className:

```tsx

  variant="primary"
  className="rounded-full px-8 shadow-lg hover:shadow-xl"
```

Most components support size variants:

```tsx
<div className="flex items-center gap-2">
  <Button size="sm">Small</Button>
  <Button size="md">Medium</Button>
  <Button size="lg">Large</Button>
</div>
```
